form action=""
form태그에 action속성은 써줘야한다.
method=""
method속성은 필요하면 써야겠다.
* action : 해당 form안에 사용자가 입력한 값들을 전달하면서 요청 보낼 URL을 작성
* method : 요청 전송 방식 지정하는 속성(GET방식(기본값) / POST방식)
> GET방식 : 요청 시 사용자가 입력한 값들이 URL에 노출
> POST방식 : 요청 시 사용자가 입력한 값들이 URL에 안보임
from태그 내부에 존재하는 submit버튼을 클릭 시
action속성에 작성한 URL로
method속성에 작성한 요청 방식을 통해
사용자가 input요소에 입력한 값들이 key=value세트로 전달
