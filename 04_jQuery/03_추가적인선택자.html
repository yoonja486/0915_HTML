* 속성선택자

선택자[속성=속성값] : 속성값이 특정값과 "일치" 하는 요소 선택 선택자[속성~=속성값] : 속성값에 특정값을 "포함"하는 요소 선택 선택자[속성^=속성값] : 속성값이 특정값으로 "시작"하는 요소 선택 선택자[속성$=속성값] : 속성값이 특정값으로 "끝"나는 요소 선택 선택자[속성*=속성값] : 속성값에 특정값이 "포함"되는 요소 선택 선탁자[속성] : 특정 속성을 가지고 있는 모든 요소 선택

* 상태 선택자

- checked 상태 선택자(radio, checkbox)

체크되어있는 입력양식 태그를 선택 #(":checked")

국적 : 한국 미국 호주

- selected 상태 선택자(select > option)

option 요소들 중 선택된(selected) 태그를 선택 $(":selected")

지역 :


선택한 지역 : 을(를) 선택하셨습니다.
